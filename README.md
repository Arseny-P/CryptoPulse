# CryptoPulse 📈

[![Vercel](https://img.shields.io/badge/Vercel-000000?style=for-the-badge&logo=vercel&logoColor=white)](https://crypto-pulse-nine-psi.vercel.app/main)

Дашборд для отслеживания курсов криптовалют в реальном времени. Проект создан с фокусом на оптимизацию работы с сетью и управление состоянием

## Основные возможности

* **Real-time обновления:** Автоматический поллинг данных (каждые 60 секунд) без перезагрузки страницы благодаря TanStack Query
* **Избранное:** Сохранение пользовательских предпочтений между сессиями (Zustand Persist)
* **Визуализация трендов:** Интегрированные sparkline-графики цены за последние 7 дней (Recharts)
* **Темная/Светлая темы:** Динамическая смена темы, синхронизированная с дизайн-токенами Chakra UI
* **Обработка состояний загрузки/ошибки:** Обработка состояний загрузки (Spinners) и ошибок API (Alerts)

## Стек технологий

* **Основа:** React, TypeScript, Vite
* **Серверный стейт и кэширование:** TanStack (React) Query
* **Клиентский стейт:** Zustand (тема, валюта, watchlist)
* **UI и стилизация:** Chakra UI (компонентная база, дизайн-токены)
* **Графики:** Recharts
* **API:** CoinGecko API
